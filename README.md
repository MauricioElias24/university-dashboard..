# University Student Dashboard

This project analyzes university student admission, enrollment, retention and satisfaction data using Python and Streamlit.

## Features

- Interactive filters
- KPI metrics
- Retention analysis
- Satisfaction analysis
- Enrollment comparison
- Department distribution charts

## Technologies Used

- Python
- Pandas
- Matplotlib
- Seaborn
- Streamlit

## Files

- app.py
- university_student_data.csv
- requirements.txt
- README.md

## Dashboard Functions

The dashboard allows users to:

- Filter data by year
- Filter data by term
- View retention trends
- Analyze student satisfaction
- Compare Spring and Fall enrollment
- View department distribution
